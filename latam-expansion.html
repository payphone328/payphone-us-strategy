Strategy

Recommended Acquiring Strategy

Three-Layer Approach

Combine cross-border, local, and orchestration for maximum coverage and best economics.

Layer 1: Cross-Border Base

Single API integration for all LATAM countries. Start processing immediately.

  • dLocal or Kushki as primary cross-border acquirer
  • Covers 15+ countries from day one
  • Includes 600+ APMs automatically
  • Higher rates but fastest time-to-market

Layer 2: Local Acquiring

Add local acquirers in priority markets for lower interchange and better approval.

  • Colombia: Wompi or Credibanco/Redeban
  • Mexico: Conekta or Openpay
  • Brazil: Stone or Cielo
  • Peru: Niubiz

Layer 3: Orchestration

Add Yuno or Spreedly to route intelligently across all acquirers.

  • Smart routing per transaction type
  • Automatic failover if one acquirer declines
  • A/B test acquirers for best approval rates
  • Unified reporting across all providers

Betting-Specific

For iGaming/betting payment processing, add specialized providers.

  • Nuvei — leading iGaming payment provider
  • PayRetailers — LATAM iGaming specialist
  • PIX integration critical for Brazil betting
  • Crypto payment support for offshore books
